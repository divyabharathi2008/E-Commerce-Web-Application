E-Commerce Web Application

Project Overview

A simple E-Commerce Web Application developed using HTML, CSS, and JavaScript. The application allows users to view products, add products to a shopping cart, remove products from the cart, and calculate the total amount dynamically.

Features

- Product Listing
- Add to Cart Functionality
- Remove from Cart Functionality
- Automatic Total Price Calculation
- Dynamic Cart Updates
- Simple User Interface

Technologies Used

- HTML
- CSS
- JavaScript

Project Structure

- index.html
- style.css
- script.js

How It Works

1. Users can view available products.
2. Click the "Add to Cart" button to add a product.
3. Added products are displayed in the cart section.
4. Total amount updates automatically.
5. Users can remove products using the "Remove" button.
